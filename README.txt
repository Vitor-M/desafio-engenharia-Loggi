Sobre o Projeto:
Linguagem utilizada: Python 3.8

Como Executar:
Basta remover os comentários que executam as funções, referentes as questões que se deseja testar. Após isto, para rodar o código, o usuário deve apenas executar em seu terminal o comando "python3 vitormanoel_desafio_engenharia.py"

Estrutura da solução do desafio:
O projeto foi divido em dois arquivos:
No arquivo principal (vitormanoel_desafio_engenharia.py) foram criadas as funções, com os algoritmos de solução das questões do desafio.
No arquivo BarcodeLoggi.py é definida a classe que representa a estrutura de código de barras apresentada no desafio.
